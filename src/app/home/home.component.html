Yu-Gi-Oh! Forbidden Memories Database
info About this project
"Yu-Gi-Oh! Forbidden Memories" is PlayStation game based on the Yu-Gi-Oh! manga and anime series.
This app aims to help players navigate the game by providing useful tools to search for powerful cards, check where to find them, explore duelists and their drops, and calculate fusion combinations.
FEATURES
style Cards
Explore the full collection of cards available in game. Find details on each card and how to obtain them.

Learn about the duelists you'll face throughout the game, including the cards they drop and the ones they use in battle.
